# Ecommerce Recommendation System
## Milestone 1: Data Collection and Preprocessing

This project  implements the first milestone of building an e-commerce recommendation engine.

### Tasks Completed
- Selected an e-commerce dataset from Kaggle
- Loaded and explored the dataset using Google Colab
- Cleaned missing and invalid values
- Converted User ID and Product ID to numeric format
- Removed invalid IDs (0 values)
- Filled missing text fields
- Created a user-item interaction matrix
- Modularized preprocessing using preprocess_data.py
- Set up a virtual environment and project structure in VS Code

### Files Included
- preprocess_data.py → Data cleaning logic
- app.py → Runs preprocessing
- requirements.txt → Project dependencies
- clean_data.csv → Cleaned dataset
- user_item_matrix.csv → User-item interaction matrix

### Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
